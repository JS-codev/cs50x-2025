Before you start on "Buy" section, you need to create a new table in SQL as it doesn't have the correct table initially. 

How set up table:

- Type sqlite3 finance.db to enter SQL.

- Paste code below:
sqlite> CREATE TABLE transactions (
   ...>     id INTEGER PRIMARY KEY AUTOINCREMENT,
   ...>     user_id INTEGER NOT NULL,
   ...>     symbol TEXT NOT NULL,
   ...>     shares INTEGER NOT NULL,
   ...>     price NUMERIC NOT NULL,
   ...>     timestamp DATETIME DEFAULT CURRENT_TIMESTAMP,
   ...>     FOREIGN KEY (user_id) REFERENCES users(id)
   ...> );

- Type `.tables` to update check if table has been added.
sqlite> .tables

- Type `.exit` to exit SQL.
